01.텍스트 요소의 종류

HTML (HyperText Markup Language) 은 웹 페이지의 형태를 기술하기 위한 마크업 언어를 의미합니다.
웹의 기초적인 구성 요소로, 웹 페이지의 구조와 콘텐츠의 의미를 정의합니다.

텍스트 요소의 종류
요소 설명 렌더링
< strong > 중대하거나 긴급한 의미의 텍스트임을 강조하기 위해 사용합니다. 굵은 글씨체
< b > 특별한 의미 없이, 단순히 다른 텍스트와의 구별 및 텍스트 자체에 주의를 끌기 위해 사용합니다. 굵은 글씨체
< em > 텍스트의 강세를 나타내며, 구어체 강조 등으로 문장의 의미를 변경하기 위해 사용합니다. 기울임꼴
< mark > 주의 깊게 볼 텍스트임을 강조하기 위해 사용합니다. 텍스트 배경 노랑 (하이라이트)
< small > 덧붙이는 글, 저작권, 법률 표기, 주의사항 등의 보조 주석을 표현하기 위해 사용합니다. 일반 텍스트보다 2포인트 정도 작은 텍스트
< sup > 위 첨자 텍스트를 표시하기 위해 사용합니다.
< sub > 아래 첨자 텍스트를 표시하기 위해 사용합니다.
